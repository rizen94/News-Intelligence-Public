# Scripts Archive

**Date**: December 2024  
**Status**: Archive Directory

---

## 📋 Purpose

This directory contains scripts that have been archived because they are:
- One-time migrations (completed)
- Duplicate versions (superseded)
- Test/debug scripts (not needed in production)
- Obsolete utilities (no longer used)
- One-time setup scripts (already completed)

---

## 📁 Archive Structure

### **v4_migrations/**
V4 migration scripts (one-time, completed)
- All `v4_*.py` and `v4_*.sh` scripts
- Database migration scripts
- Architecture update scripts

### **duplicate_production/**
Duplicate production scripts (superseded by main scripts)
- Multiple versions of start/stop scripts
- Duplicate GPU setup scripts
- Redundant deployment scripts

### **one_time_setup/**
One-time setup scripts (already completed)
- System setup scripts
- Configuration scripts
- Initial setup utilities

### **test_scripts/**
Test and debug scripts (development only)
- Integration tests
- Debug utilities
- Test pipelines

### **obsolete/**
Obsolete scripts (no longer used)
- Old utilities
- Unused maintenance scripts
- Legacy scripts

### **deployment/**
Deployment-specific scripts (may not be actively used)
- Hardware deployment
- Template scripts
- Deployment dashboards

---

## 🔍 Finding Active Scripts

For currently active scripts, see:
- `../` (scripts root) - Active production scripts
- `../production/` - Production management scripts
- `../maintenance/` - Maintenance utilities
