## **🛠️ Development Workflow**

### **Making Changes**
1. **Edit Schema**: Update `schema/unified_schema.json`
2. **Regenerate Code**: Run `python3 scripts/generate_from_schema.py`
3. **Test System**: Run `python3 test.py`
4. **Deploy**: Run `./start.sh`

### **Production Deployment**
1. **Start System**: `./start.sh`
2. **Monitor Health**: `python3 test.py`
3. **Stop System**: `./stop.sh`
